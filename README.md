# Liuyang-Fireworks-Data-Analysis
🎇 浏阳烟花受众满意度及消费决定因素分析
📊 项目概述
本项目基于 1027 份真实问卷数据，利用 Python 进行了深度数据清洗、受众画像刻画，并构建了 Logistic 回归模型探讨消费转化核心驱动力。

🔍 核心洞察 (Highlights)
高转化路径：线下节庆体验是受众的首要触点，留存转化率达 XX%。

量化结论：(等我们做完第二节再填，比如：文化认同感每提升 1 单位，付费意愿增加 X%)。

🛠 技术栈
数据处理: Pandas, NumPy

可视化: Matplotlib, Seaborn

统计建模: Statsmodels (Logistic Regression)
