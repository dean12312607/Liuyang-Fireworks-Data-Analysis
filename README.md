# 🎇 浏阳烟花受众满意度及消费决定因素深度分析
> 本项目基于 1027 份真实问卷数据，利用 Python 进行端到端的数据清洗、探索性分析（EDA），并构建高阶统计学模型，旨在为非遗文化的商业化落地提供量化支撑与精准营销策略。

## 📁 核心工程目录 (Project Structure)
- `data/` : 存放清洗后的脱敏分析数据 (CSV格式)。
- `notebooks/` : 存放核心代码，按章节模块化拆分。
  - `Day1_数据清洗.ipynb`: 异常值处理、数据规整。
  - `Day2_EDA_受众画像.ipynb`: AARRR转化漏斗、多选题拆解、交叉热力图。
  - `Day3_Logistic建模.ipynb`: 向后剔除法特征工程、ROC模型评估、异质性分组检验。
- `output/` : 存放所有由代码自动生成的高清可视化商业图表。

## 📊 阶段性核心洞察 (Key Business Insights)

### 第一阶段：受众画像与消费现状
- **漏斗转化极佳**：受众从“知晓”到“体验”再到“核心付费”呈现出极其健康的转化漏斗（精准意愿转化率超 46%）。
- **核心触点在线下**：多选题扁平化处理后发现，“线下节庆实地观赏”是建立受众认知的第一渠道，短视频传播紧随其后。

### 第二阶段：付费意愿的 Logistic 回归建模
- **严谨的统计学验证**：模型满意度量表 Cronbach's Alpha 达 0.964，二分类模型全局预测准确率达 67.9%（AUC曲线优良）。
- **颠覆认知的驱动力**：利用向后剔除算法从 27 个变量中提纯出 5 大核心驱动力。传统的“色彩/造型”不具备显著溢价能力，**“环保指标”与“创意表现”**才是决定掏钱的核心。
- **人群异质性“双标”**：
  - **Z世代 (25岁以下)**：为“环保”和“KOL种草”买单，对单纯的创意脱敏。
  - **中青年 (26岁以上)**：对环保不敏感，只为震撼的“创意”和“非遗文化情怀”买单。
